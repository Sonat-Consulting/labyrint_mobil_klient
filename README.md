# Fagdag labyrint mobil klient

## start app i emulator

- (cmd+shift+p) => Flutter: Launch Emulator
- Flutter run

## build android

- flutter clean
- flutter build apk
- kopier fil til telefon / installer ved å klikke på apk filen

## Build ios

- flutter clean
- flutter build ios
- open ios/Runner.xcworkspace
